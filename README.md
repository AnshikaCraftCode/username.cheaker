## ğŸš€ How to Use

1. **Open the Application**
   - Launch `index.html` in your web browser

2. **Enter a Username**
   - Type your desired username in the input field
   - Username should be 3-20 characters long

3. **Click the "Check" Button**
   - Click the Check button to validate the username

4. **View the Result**
   - **Available** (Green): Username is available for use
   - **Taken** (Red): Username is already in use
   - **Invalid** (Yellow): Username doesn't meet requirements
